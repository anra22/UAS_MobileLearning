PICTURE

Materi :

Kolom ini digunakan untuk manambahkan gambar.

1. Isikan Field Name

3. Isi width in single view untuk menampilkan gambar pada tampilan single view

6. Isi height in list view untuk menampilkan gambar pada tampilan list view.

7. Pilih ukuran maksimum gambar pada maximum size
